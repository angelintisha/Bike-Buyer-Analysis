# Bike Buyers Analysis — Excel Dashboard

> Excel skills demonstration: data cleaning, pivot tables, and interactive dashboard
> analyzing customer demographics and bike purchase behavior.

## Overview
An end-to-end Excel project analyzing 1,000 customer records to identify demographic 
patterns among bike buyers. The dashboard allows filtering by marital status, region, 
and education to explore which customer segments are most likely to purchase a bike.

## Tech Stack
![Excel](https://img.shields.io/badge/Microsoft-Excel-217346?logo=microsoft-excel&logoColor=white)

## Data Structure
**bike_buyers** — 1,000 rows · 13 columns

| Column | Description |
|--------|-------------|
| ID | Customer identifier |
| Marital Status | Married or Single |
| Gender | Male or Female |
| Income | Annual income (USD) |
| Children | Number of children |
| Education | Education level |
| Occupation | Job category |
| Home Owner | Yes or No |
| Cars | Number of cars owned |
| Commute Distance | Distance to work |
| Region | Europe, Pacific, North America |
| Age | Customer age |
| Purchased Bike | Target variable — Yes or No |

## Dashboard Features
- **Average Income by Gender** — comparison between bike buyers and non-buyers
- **Commute Distance** — purchase likelihood by distance to work
- **Age Bracket** — purchase trends across age groups
- **Interactive Slicers** — filter by marital status, region, and education level

## Key Findings
- Middle-aged customers (31–54) purchase bikes at the highest rate
- Higher income correlates with bike purchases across both genders
- Customers with shorter commutes (0–1 miles) are more likely to purchase
- Male buyers have a higher average income than female buyers across both groups
